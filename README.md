# docscss-issues
Welcome to the issue tracker for the Help Scout Docs CSS generator found at https://itcouldbenothing.com/docscss. 

This repository is dedicated to tracking bugs, feature requests and feedback for this private project. There’s no code in this repository — it exists only as a place to file and discuss issues.

If you have suggestions, run into problems or want to request a new feature, please open an issue using the templates provided.

## What you can do here
- Report bugs  
- Request new features  
- Ask questions related to the Help Scout Docs CSS Generator  
- Share feedback

## What you cannot do here
- Access the source code  
- Submit pull requests  
- Fork this repository to contribute code

## How to file an issue
1. Click **New Issue**.  
2. Choose the appropriate template.  
3. Fill out the details so I can understand the problem or request.

Note that no support is offered for Help Scout or CSS in general here and bugs and feature requests will be handled as time allows.
